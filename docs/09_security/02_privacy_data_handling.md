# 02. 개인정보/민감정보 취급(초안)

## 1) 위치 정보
- 기본: 동네 단위 저장
- 정밀 좌표: 기능상 필요한 경우(산책 코스/실종 제보)만 저장
- 노출: 사용자에게 “노출 범위” 옵션 제공(동네/반경)

## 2) 영수증/사진
- 영수증 업로드는 선택
- 업로드 전 마스킹 안내(이름/전화번호 등)
- 사용자 삭제 기능 제공
- 보관 기간 정책(예: 1년 또는 사용자 요청 시 즉시 삭제)

## 3) 연락처 노출
- 기본은 플랫폼 내 문의(알림) 권장
- 외부 연락처/링크는 단계적으로 허용(스팸 통제)

## 4) 데이터 비즈니스(집계)
- 개인 식별 불가능한 집계/세그먼트 형태만 제공
- 동의/거부 UI 제공
- 투명성 문서(어떤 데이터를 어떤 목적으로 사용하는지) 유지
