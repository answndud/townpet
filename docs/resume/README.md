# Sionic AI Native Engineer Fellowship 지원 패키지

목표: TownPet을 "AI로 만든 개인 프로젝트"가 아니라 "AI-native 방식으로 문제 정의, 구현, 검증, 배포, 운영까지 닫은 엔지니어링 증거"로 제출한다.

## 파일 구성

1. [strategy.md](./strategy.md)
   - 공고 해석, 평가 기준, TownPet 증거 매핑, 리스크와 방어 논리
2. [application-draft.md](./application-draft.md)
   - 제출폼에 바로 옮길 자기소개, 지원 이유, AI 활용 경험, 포트폴리오 설명
3. [interview-playbook.md](./interview-playbook.md)
   - 30초/3분 피치, 대표 사례 답변, 공격 질문 방어, 데모 순서
4. [submission-checklist.md](./submission-checklist.md)
   - 제출 직전 확인할 링크, 문장, 금지 표현, 최종 점검표

## 핵심 포지셔닝

> 저는 AI를 사용해 코드를 빠르게 쓰는 사람보다, AI와 함께 문제를 쪼개고 결과를 검증하며 운영 가능한 형태까지 닫는 엔지니어입니다.
