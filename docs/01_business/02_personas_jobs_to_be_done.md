# 02. 페르소나 & JTBD

## 1) 페르소나 1: 초보 견주(동네 생활 적응)
- 상황: 첫 반려견, 병원/산책/훈련 정보가 부족
- 목표: “잘못 선택해서 비용/시간을 날리고 싶지 않다”
- JTBD
  - 병원비/대기/서비스 정보를 확인하고 방문 결정을 내린다
  - 안전한 산책 코스를 찾고 위험 요소를 회피한다
  - 사료/알러지 관련 질문을 빠르게 해결한다

## 2) 페르소나 2: 다견/다묘 가구(반복 비용 최적화)
- 상황: 병원 방문 잦고 사료/모래 구매량이 큼
- 목표: “반복 비용을 줄이고, 품질 있는 공급원을 확보”
- JTBD
  - 항목별 병원비 분포를 보고 합리적 선택을 한다
  - 공동구매로 배송비/단가를 낮춘다
  - 대여/중고로 장비 구매 부담을 줄인다

## 3) 페르소나 3: 긴급 돌봄이 필요한 직장인
- 상황: 야근/출장/갑작스러운 일정으로 돌봄 공백 발생
- JTBD
  - 단시간 펫시팅/산책/심부름을 가까운 이웃에게 요청한다
  - 신뢰 가능한 사람을 빠르게 선별한다(평판/인증/후기)

## 4) 페르소나 4: 도그워커/펫시터(공급자)
- 상황: 소규모로 일감을 꾸준히 확보하고 싶음
- JTBD
  - 동네 기반 요청을 받아 스케줄을 채운다
  - 신뢰도(후기/완료율)를 통해 단가를 안정적으로 만든다

## 5) 페르소나 5: 지역 업체(병원/미용/카페)
- 상황: 인스타/네이버 외에 로컬 고객이 필요
- JTBD
  - 지역 타겟 노출로 유입을 만든다
  - 쿠폰/이벤트를 발행해 재방문을 유도한다

## 핵심 플로우(요약)
- Local: 동네 설정 → 병원/장소/산책 탐색 → 템플릿 리뷰 작성 → 저장/알림
- Market: 등록 → 문의 → 거래/대여 상태 변경 → 후기
- Care(마이크로 시팅): 요청 → 매칭 → 수행 → 정산/후기
- Lost&Found: 신고 → 위치 기반 푸시 → 제보/발견 → 종료/감사
- Global: 질문 → 답변 → 채택 → 지식 축적/검색 유입
