# 01. IA / 내비게이션 설계(Local vs Global 분리)

## 1) 상위 탭(권장)
- 홈(기본: Local 요약)
- 동네(Local)
- 지식(Global)
- 마켓(거래 중심: Local, 정보 중심: Global 연결)
- 돌봄/심부름(요청/매칭)
- 알림
- 프로필/설정

## 2) 홈 정책
- 기본은 Local 하이라이트(내 동네 최신/인기, 긴급 알림 배너)
- Global은 “추천 토픽/인기 Q&A” 위젯 형태로 최소 노출
- 홈을 하나의 무한 피드로 섞지 않는다(혼탁 방지)

## 3) Local 영역(동네)
### 카테고리
- 병원/동반가능 장소(식당/테라스 카페 등)
- 산책로(지도)
- 산책 메이트 & 펫친소(번개/프로필 탐색)
- 마켓·대여·나눔(지역 거래)
- 심부름/긴급 돌봄(마이크로 펫시팅)
- 실종/유기(긴급)

### 핵심 UX
- 지도/거리/동네 스코프가 기본
- 템플릿 기반 작성(구조화)
- 위치 기반 푸시(긴급) 및 키워드 알림

## 4) Global 영역(지식)
### 카테고리
- 품종별 토픽(견종/묘종)
- Q&A(답변 채택)
- 사료/용품 DB(성분/알러지 필터 + 리뷰)
- 전문가 콘텐츠(가이드/칼럼)

### 핵심 UX
- 검색/태그/채택/업보트 기반
- 중복 질문 방지(유사 질문 추천)
- 전문가 콘텐츠는 “출처/자격/면책”을 명확히

## 5) 전역 검색
- Local/Gobal 탭 토글
- Local: 동네/거리/최근성 우선
- Global: 토픽/유용성(업보트/채택) 우선
