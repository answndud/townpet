# 02. 기능 카탈로그(전체 기능 목록)

> 표의 우선순위는 “Phase”로 표현하며, 실제 구현 순서는 `03_mvp_scope_v1.md`를 따른다.

## 1) 공통(Core)
| 기능 | 설명 | Phase | 의존성 |
|---|---|---:|---|
| 회원/프로필 | OAuth2 로그인, 닉네임, 반려동물 프로필 | 1 | Auth |
| 동네 설정 | 대표 동네/인접 동네, 동네 인증 레벨 | 1 | Geo |
| 게시물/댓글 | CRUD, 대댓글, 좋아요/저장 | 1 | DB |
| 신고/차단 | 신고 사유, 차단, 자동 블라인드 | 1 | Moderation |
| 검색/필터 | Local/Global 토글, 태그/정렬 | 1 | Search |
| 알림 | 댓글/저장키워드/긴급 알림 | 2 | Notification |
| 관리자 콘솔 | 신고 큐, 제재, 로그 | 2 | Admin |

## 2) Local(동네)
| 기능 | 설명 | Phase | 의존성 |
|---|---|---:|---|
| 병원 리뷰 템플릿 | 방문일/항목/비용/대기/영수증 | 1 | Place DB |
| 동반가능 장소 | 식당/테라스/카페, 조건 태그 | 1 | Place DB |
| 산책로 공유 | 지도 기반 코스/안전 태그 | 1 | Map |
| 산책 메이트 번개 | “지금 공원 오실 분?”, 참여/채팅(후순위) | 2 | Notification |
| 펫친소 프로필 탐색 | 동네 친구 탐색, 성향 태그 | 2 | Profile |
| 마켓(중고) | 등록/문의/상태(판매중/예약/완료) | 2 | Moderation |
| 대여 | 보증금/기간/인수·반납 체크리스트 | 2 | Market |
| 나눔 | 무료 나눔, 우선순위 룰(선착/선택) | 2 | Market |
| 마이크로 펫시팅 | 긴급 돌봄/심부름 요청·매칭 | 3 | Payment(선택) |
| 실종/유기 긴급 | 위치 기반 푸시, 제보/종료 | 2~3 | Push, Geo |

## 3) Global(지식)
| 기능 | 설명 | Phase | 의존성 |
|---|---|---:|---|
| 품종별 토픽 | 견종/묘종별 게시판 | 2 | Taxonomy |
| Q&A + 채택 | 답변 채택, 업보트/정렬 | 2 | Search |
| 사료/용품 DB | 성분/알러지 필터, 리뷰 축적 | 3 | Data model |
| 전문가 콘텐츠 | 수의사/훈련사 기고, 검수/면책 | 3 | Policy |

## 4) BM(수익화) 기능
| 기능 | 설명 | Phase | 의존성 |
|---|---|---:|---|
| 상단 노출/광고 | 지역 업체 상단 노출, 쿠폰 | 3 | Billing |
| 돌봄 수수료 | 예약/정산/수수료 | 4 | Payment, Dispute |
| 공동구매 | 모집/마감/결제/배송 | 4 | Payment, Ops |
| 제휴 링크 | 추천→구매 링크 | 3 | Product DB |
| 멤버십 | 광고 제거, 상담권, 특가 | 4 | Billing |
| 데이터 리포트 | 집계 리포트 판매 | 5 | Legal/Privacy |
