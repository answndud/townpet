# community-taxonomy-v1.md
# 멀티종 커뮤니티 분류/운영 초안 (v1)

기준일
- 2026-02-25

문서 목적
- TownPet를 반려견/반려묘 중심에서 멀티종 커뮤니티로 확장하기 위한 초기 고정 커뮤니티 구성을 정의한다.
- 한국형 커뮤니티 사용성에 맞춰 멤버십(가입/구독) 없이 탐색/참여 가능한 구조를 확정한다.

핵심 원칙
- 멤버십 기능은 MVP 범위에서 제외한다.
- 커뮤니티 목록은 운영자가 사전 정의한다(초기 사용자 생성 커뮤니티 없음).
- 게시글은 `커뮤니티 전용` 또는 `공용 보드` 중 1개 보드에 소속된다.
- 기존 안전 정책(신규유저 제한, 링크/연락처 제한, 신고 임계 자동숨김)을 전 커뮤니티에 동일 적용한다.

---

## 1) 상위 분류(L1)

초기 대분류는 사용자가 즉시 이해 가능한 동물군 기준으로 고정한다.

1. 강아지
2. 고양이
3. 조류
4. 파충류
5. 소동물
6. 어류/수조
7. 양서류
8. 절지류/곤충
9. 특수동물/기타

운영 규칙
- L1은 탐색 필터와 홈 디렉토리 진입용으로 사용한다.
- L1 단위 게시판은 만들지 않고, 실제 활동 단위는 L2 커뮤니티로 운영한다.

---

## 2) 초기 고정 커뮤니티(L2) 12개

초기 오픈 커뮤니티는 대중 트래픽과 매니아 수요 균형 기준으로 아래 12개를 고정한다.

| 표시명 | slug | 대분류(L1) | 한줄 설명 |
|---|---|---|---|
| 강아지 | `dogs` | 강아지 | 반려견 일상, 건강, 훈련, 용품 정보를 공유하는 기본 커뮤니티 |
| 고양이 | `cats` | 고양이 | 반려묘 일상, 행동, 케어, 용품 정보를 공유하는 기본 커뮤니티 |
| 조류 | `birds` | 조류 | 조류 전반 사육, 건강, 환경 정보를 다루는 통합 커뮤니티 |
| 앵무새 | `parrots` | 조류 | 앵무새 종 중심의 훈련, 소통, 케이지/장난감 운영 정보 공유 |
| 파충류 | `reptiles` | 파충류 | 파충류 전반 입문/사육/환경 세팅 정보를 다루는 통합 커뮤니티 |
| 도마뱀 | `lizards` | 파충류 | 도마뱀 종 중심의 온습도, 먹이, 사육장 관리 노하우 공유 |
| 뱀 | `snakes` | 파충류 | 뱀 종 중심의 사육 안정화, 급이, 핸들링 이슈를 다루는 커뮤니티 |
| 거북 | `turtles` | 파충류 | 수생/육상 거북의 여과, 일광욕, 성장관리 정보를 다루는 커뮤니티 |
| 소동물 | `small-pets` | 소동물 | 햄스터/토끼/고슴도치 등 소형 반려동물 통합 커뮤니티 |
| 어류·수조 | `aquatics` | 어류/수조 | 관상어/수초/수질/여과기 세팅을 중심으로 운영하는 커뮤니티 |
| 양서류 | `amphibians` | 양서류 | 개구리/뉴트/도롱뇽 사육 및 환경 관리 정보를 공유하는 커뮤니티 |
| 절지류·곤충 | `arthropods` | 절지류/곤충 | 타란툴라/전갈/사마귀 등 절지류 사육 정보를 공유하는 커뮤니티 |

표기 규칙
- UI 표시명은 한국어 우선, URL/내부키는 영문 slug 사용.
- slug는 소문자 + 하이픈만 허용한다.

## 3) 공용 보드(전 커뮤니티 참여) 정책

아래 카테고리는 커뮤니티를 나누지 않고 공용으로 운영한다.

| 공용 보드 | 연결 PostType | 운영 이유 | 노출 원칙 |
|---|---|---|---|
| 병원후기 | `HOSPITAL_REVIEW` | 동물종보다 지역/병원 정보 가치가 높음 | 지역 우선 + 동물 태그 보조 |
| 실종/목격 제보 | `LOST_FOUND` | 긴급성과 도달률이 핵심이라 분리 시 손해 | 긴급 고정 상단 + 지역 우선 |
| 중고/공동구매 | `MARKET_LISTING` | 거래 유동성/매칭률 확보가 중요 | 최신/신뢰 점수 + 동물 태그 필터 |

카테고리-보드 매핑 규칙
- `HOSPITAL_REVIEW`, `LOST_FOUND`, `MARKET_LISTING`는 `COMMON` 보드에서만 작성 허용.
- `FREE_BOARD`, `QA_QUESTION`, `PRODUCT_REVIEW`, `PET_SHOWCASE`는 `COMMUNITY` 보드에서 작성.
- 공용 보드 글은 커뮤니티 소속 대신 `animalTags`(예: 강아지, 고양이, 파충류)로 탐색성을 확보.

전체 `PostType` 매핑표

| PostType | 기본 보드 | 비고 |
|---|---|---|
| `HOSPITAL_REVIEW` | `COMMON` | 공용 보드: 병원후기 |
| `LOST_FOUND` | `COMMON` | 공용 보드: 실종/목격 제보(긴급 우선) |
| `MARKET_LISTING` | `COMMON` | 공용 보드: 중고/공동구매 |
| `FREE_BOARD` | `COMMUNITY` | 커뮤니티 기본 자유글 |
| `QA_QUESTION` | `COMMUNITY` | 커뮤니티 질문 |
| `PRODUCT_REVIEW` | `COMMUNITY` | 커뮤니티 용품리뷰 |
| `PET_SHOWCASE` | `COMMUNITY` | 커뮤니티 반려자랑 |
| `PLACE_REVIEW` | `COMMUNITY` | 초기에는 커뮤니티 보드로 유지(후속 공용 전환 가능) |
| `WALK_ROUTE` | `COMMUNITY` | 초기에는 커뮤니티 보드로 유지(후속 공용 전환 가능) |
| `MEETUP` | `COMMUNITY` | 커뮤니티 기반 모임(정책 강화) |
| `FREE_POST` | `COMMUNITY` | 레거시 타입, `FREE_BOARD`와 동일 그룹 |
| `DAILY_SHARE` | `COMMUNITY` | 레거시 타입, `FREE_BOARD`와 동일 그룹 |
| `QA_ANSWER` | `COMMUNITY` | 레거시 타입, `QA_QUESTION`과 동일 그룹 |

데이터 모델 메모(초안)
- `Post.boardScope`: `COMMUNITY | COMMON`
- `Post.petTypeId`: `boardScope=COMMUNITY`일 때 필수 (`@map("communityId")`로 DB 컬럼 호환)
- `Post.commonBoardType`: `boardScope=COMMON`일 때 필수 (`HOSPITAL_REVIEW | LOST_FOUND | MARKET_LISTING`)
- `Post.animalTags`: 공용 보드 글의 보조 탐색 태그

운영 메모
- 공용 보드는 기존 정책 대비 더 강한 안전장치를 권장한다(신규유저 제한 강화, 연락처/외부링크 제한 강화, 신고 임계 낮춤).
- 실종/목격 제보는 `LOCAL` 우선 노출을 기본값으로 유지하되, `GLOBAL` 탐색도 허용한다.

- `자유게시판` = `FREE_BOARD`
- `질문/답변` = `QA_QUESTION`
- `반려자랑` = `PET_SHOWCASE`

## 6) 초기 오픈 제외(후속 분화 후보)

아래 커뮤니티는 초기 과분화를 피하기 위해 통합 운영 후 데이터 기준으로 분화한다.

- 페럿(`ferrets`)
- 친칠라(`chinchillas`)
- 기니피그(`guinea-pigs`)
- 고슴도치(`hedgehogs`)
- 토끼(`rabbits`)
- 해수어/마린(`marine-aquarium`)

분화 기준(권장)
- 2주 연속 주간 글 30개 이상 + 고유 작성자 15명 이상일 때 분리 검토.

통합 기준(권장)
- 4주 연속 주간 글 10개 미만일 때 상위 통합 커뮤니티로 병합 검토.
