# 02. 모더레이션(신고/제재) 규칙

## 1) 신고 사유(카테고리)
- 허위정보/조작
- 광고/스팸
- 비방/명예훼손
- 사기/금전 요구
- 불법 구인/불법 행위
- 개인정보 노출
- 기타(운영자 판단)

## 2) 자동 블라인드 룰(초안, 운영하면서 튜닝)
- 동일 게시물 신고 `>= 3` : 임시 숨김(HIDDEN) + 작성자에게 알림
- 동일 사용자 최근 7일 신고 누적 `>= 5` : 글쓰기 24시간 제한
- 신규 계정(가입 24시간 이내) 신고 `>= 2` : 기능 제한 강화(마켓/구인/긴급 신고 불가)

## 3) 재노출(복구) 기준
- 허위/비방이 아닌 것으로 판단되면 복구
- 템플릿 필드 보완(방문일/항목/금액 등)이 완료되면 복구 가능
- 반복 위반자는 복구 기준을 강화

## 4) 실종/유기 “긴급” 남용 방지
- 긴급 글은 제한 조건(동네 인증/가입 기간/활동 점수) 중 최소 1개를 만족해야 함
- 긴급 글에는 운영자 “신속 검토” 큐를 별도로 운영

## 5) 관리자 액션 로그(Audit)
- 조치 종류: HIDE/UNHIDE/BAN/TEMP_BAN/WARN/EDIT_REQUEST
- 필수 기록: 대상, 사유, 근거(신고 수/로그), 처리자, timestamp
