# 02. 가격/수익화 실험 설계(초안)

## 1) 부스팅(광고) 실험
- 대상: 로컬 업체 상단 노출
- 실험: 노출 단위(주간/월간), 지역 범위, 카테고리별 단가
- 측정: 클릭률(프로필/길찾기/전화), 쿠폰 사용률

## 2) 멤버십 실험
- 대상: 고빈도 사용자(저장/검색/작성 높은 사용자)
- 혜택 A/B
  - A: 광고 제거 + 고급 알림
  - B: 전문가 상담권 + 특가
- 측정: 전환률, 해지율, ARPU

## 3) 제휴 링크 실험
- 사료/용품 추천 시나리오에서 링크 노출 위치 테스트
- 측정: 클릭률, 구매 전환률, 반품/불만
