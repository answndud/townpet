# 01. 로드맵(Phase 기반)

## Phase 1: Local UGC 코어
- 병원/동반장소 템플릿 리뷰
- 산책로 지도/태그
- 동네 설정, 게시물/댓글/저장, 신고/블라인드
- 최소 검색

## Phase 2: 로컬 네트워크 강화 + 마켓
- 번개/펫친소
- 마켓/대여/나눔 + 상태 전이
- 실종/유기(긴급) + 기본 알림

## Phase 3: Global 지식 + 돌봄 요청
- 품종 토픽, Q&A 채택
- 사료/용품 DB(기본)
- 마이크로 돌봄 요청/매칭
- 제휴 링크

## Phase 4: 수익화 고도화
- 부스팅/광고
- 멤버십
- 공동구매
- 돌봄/거래 수수료 + 결제/정산

## Phase 5: 데이터/파트너십
- 보험/제조사 리포트(집계)
- 전문가 네트워크 확장
