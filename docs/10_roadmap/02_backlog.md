# 02. 백로그(우선순위 후보)

## P1 (필수)
- 커서 페이지네이션 피드
- 템플릿 필드 강제(병원/장소/산책)
- 신고/블라인드 자동화
- 관리자 신고 큐
- 지도 핀/코스 표시

## P2 (중요)
- 번개 모임(참여/취소/노쇼)
- 마켓(대여 체크리스트)
- 실종/유기 푸시(반경/동네)
- 키워드/태그 구독 알림

## P3 (확장)
- Q&A 채택/업보트
- 사료/용품 DB(성분/필터)
- 돌봄 요청/매칭(결제 없이)

## P4 (수익화/운영 비용 상승)
- 결제/정산/환불
- 광고/부스팅/쿠폰
- 공동구매(배송/환불)
- 전문가 유료 상담

## P5 (나중)
- 실시간 채팅
- 추천 알고리즘 고도화(개인화)
- 멀티 지역 확장(도시별 운영)
